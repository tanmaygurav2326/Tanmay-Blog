Solving M1: Automating Determinants with C

First-year mathematics can feel like a boss battle. So I decided to automate determinant calculation using C. 


#include 

int main() {

int a,b,c,d,determinant;

printf("Enter values:");
scanf("%d %d %d %d",&a,&b,&c,&d);

determinant=(a*d)-(b*c);

printf("Determinant = %d",determinant);

return 0;

}

Coding math formulas makes them much easier to remember. Next goal: automate a 3×3 matrix determinant.
